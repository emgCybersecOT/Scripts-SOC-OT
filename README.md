# Scripts-SOC-OT

Repositorio centralizado de herramientas y scripts de automatización desarrollados en Python para el soporte de operaciones en un Centro de Operaciones de Seguridad (SOC) enfocado en entornos industriales (OT) e infraestructura crítica.

El objetivo de este laboratorio es optimizar la monitorización, el filtrado de eventos de red y el análisis de logs procedentes de activos de planta, estableciendo puentes de eficiencia entre el desarrollo de software (IT) y la automatización industrial (OT).

## Índice de Scripts Disponibles

- Script_base_pruebas.py: Script base que simula el filtrado de registros de planta (PLC, Robótica, SCADA) correlacionando bucles y condicionales de Python con la lógica de programación de autómatas bajo el estándar IEC 61131-3.



## Tecnologías y Entornos

- Lenguaje principal: Python 3.x
- Ámbito de aplicación: Seguridad perimetral OT, monitorización industrial y convergencia IT/OT con sistemas de automatización.
